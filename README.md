# NESd

A NES Emulator written in Dart and Flutter.
Currently only supports macOS.

## Features

- CPU, PPU, and APU emulation
- SRAM saves
- Save states
- Customizable keyboard controls

## Supported mappers

- NROM (0)
- MMC1 (1)
- UNROM (2)
- CNROM (3)
- MMC3 (4)
- AxROM (7)
- BR909x (71)
